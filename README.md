# PCB Power Supply Design

An AC to DC power supply PCB designed from scratch, printed, and soldered successfully.

## Overview
Converts 220V AC to 5V DC using a transformer, rectifier, and voltage regulator.

## Components
- Transformer (220V to 12V)
- Capacitors (1000µF and 470µF)
- Voltage Regulator (7805)
- Rectifier Diodes
- RGB LED (power indicator)
- Rosetta Board
## Tools
- EasyEDA (Schematic & PCB Design)
- PCB Printing & Soldering

## Result
PCB printed, soldered, and tested successfully — LED lit up!
